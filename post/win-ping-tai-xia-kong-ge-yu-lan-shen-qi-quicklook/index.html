Win平台下空格预览神器QuickLook

· 2017-08-15 · # QuickLook # 快速预览工具 # 软件

有用过MacOS系统的人应该都会对空格键快速预览的操作感到羡慕，按下空格即可预览文件的内容，而不必像win系统一样需要先等待软件启动加载，可以说这个功能真的非常的高效且实用。

学生时期受制于资金压力，曾有羡慕过那些可以用上MacOS的人，偶然一次机会让我知道QuickLook的时候，那感觉就像遇到一生的知己一样。

从学生时期到现在经历过数年时光，系统重装过一次又一次，系统里的软件有的用不上了，有的不想用了，唯独QuickLook跟随至今。

QuickLook基于Fluent Design规范设计，界面简洁美观，是一款开源且免费的快速预览的软件，有着和MacOS系统复刻程度极高的的快速预览功能，只需要选中想要预览的文件按下空格你就可以快速的预览图片、视频、音频、文档、代码、压缩包等主流的文件格式内容，体验感极佳。

作为IT从业人员，每天都要处理太多的图片和文档，很多时候其实并不是想要对图片或者文档进行编辑操作，但是通常win系统里想要查看图片或者文档那就免不了要打开它们，这期间不仅需要等待软件加载还会增加电脑的运行负担，而恰巧工作中的电脑性能都不需要多言语，知道的都知道！

所以一款可以空格预览空格退出或者Esc退出的软件真的可以大大的减少工作中不必要等待的时间，提高工作效率。

当然QuickLook并不是100%的完美，毕竟不是原生，所以很多格式的文件还都只能看到文件夹的大小和最后打开时间，看不到文件夹里包含了哪些内容，不过这也不是不可以解决的，如果想要支持预览更多的格式只需要安装插件即可。

热键和按钮

[ Spacebar ] 显示/隐藏预览窗口

[ Esc ] 隐藏预览窗口

[ Enter ] 打开/执行当前文件

[ Mouse ] [ ↑ ] [ ↓ ] [ ← ] [ → ]预览另一个文件

[ Mouse Wheel ] 放大缩小字体功能 放大缩小字体功能

[ Ctrl ] + [ Mouse Wheel ] 放大缩小字体功能 放大缩小字体功能

[ Wheel ] 增加/减少量
